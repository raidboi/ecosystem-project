# ecosystem-project

Here's Ecosystem Explorer! The game has 3 rounds:

Sort by Biome — match organisms to their home ecosystem (rainforest, grassland, deep ocean). 
Sort by Trophic Level — place species into the food chain (producer → primary → secondary consumer)
Keystone & Invasive Species — categorize organisms by their ecological role

Each correct drop earns 100 points, wrong placements cost 20, and every answer includes a real ecological explanation as feedback — which aligns perfectly with what you wrote in the worksheet about the game teaching players why a placement is right or wrong.
All you need to do now is to click on index.html to play
